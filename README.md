Реализация алгоритма сжатия и восстановления данных методом Хаффмана в рамках дисциплины "Алгоритмы и структуры данных" в Санкт-Петербургском политехническом университете Петра Великого.

В проекте используются собственные реализации структур данных вместо стандартных контейнеров STL.

**Язык:** C++ 11

### 1. Входные и выходные данные

- `file.txt` входной файл с исходным текстом.
- `encodedFile.txt` закодированный файл.
- `frequencyTable_encodedFile.txt` таблица частот символов.
- `decodedFile.txt` декодированный файл.

---

### 2. Запуск проекта

Для работы программы необходим файл `file.txt` с исходным текстом в корневой директории.

1. Клонирование репозитория:
```bash
git clone https://github.com/dfnabiullin/spbpu-huffman-coding
cd huffman-coding
```

2. Компиляция:
```bash
g++ -std=c++11 main.cpp PriorityQueue.cpp HuffmanNode.cpp -o huffman
```

3. Запуск:
```bash
./huffman
```

Программа выполнит кодирование исходного файла в `encodedFile.txt` (используя таблицу частот), а затем декодирует его обратно. В результате содержимое `decodedFile.txt` должно быть идентично исходному `file.txt`.
